    • About This Web-App

    LessSketchy queries the latest Craigslist apartment postings from the Bay area. It then analyzes its listing price, the presence of specific information, and some textual hints to provide a fraud assessment of the postings.

    The fraud detection model is trained with the balanced random forest algorithm using a highly imbalaned training set (99% legitimate and 1% fraudulent posts).
