    • About This Web-App

    • Queries the latest Craigslist apartment postings from the Bay area.

    • Analyzes listing price, absence of specific information, number of words and fraction of capital letters used in the posts.

    • Trained detection model with the balanced random forest algorithm for a highly imbalanced training set (99% legitimate and 1% fraudulent posts).


    Cross-validation

    Precision 74%
    Recall 80%
    False Positive 28%
    F-score 77%
